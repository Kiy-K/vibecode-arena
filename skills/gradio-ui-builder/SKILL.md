# gradio-ui-builder

Use this workflow for UI-building challenges:

1. Read `/context/challenge.md` and the latest `/context/instructions/*.md`.
2. Create the smallest runnable artifact in `/workspace`.
3. Prefer clear component structure, responsive layout, accessible labels, and stable dimensions.
4. If a command is available, run a quick syntax or smoke check before reporting completion.
5. In the final response, list changed files and the command result.
